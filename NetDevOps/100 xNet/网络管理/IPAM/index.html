    IPAM

    IP地址作为最重要的网络资源之一，与业务息息相关。随着IT建设规模日益庞大，使用的IP地址数据也越来越多，经常面临IP管理分散、私网乱建、非法接入等问题， 而传统人工维护 IP地址管理工作效率低下、统计困难。

    IPAM提供IP地址全生命周期管理解决方案，实现从规划、申请、审批、分配、变更、回收、采集、告警、备案的IP地址全视角管理，帮助企业加强IP信息准确性， 提升网络运维效率，保障网络安全，实现IP地址集中化、自动化、规范化管理。

    XX IPAM是完全自主研发的国产品牌，拥有多项专利及软件著作权。实施周期快、部署便捷，支持云服务器，支持高可用，系统稳定安全。

    1. 常见问题¶

    1. 缺乏有效地址规划
    2. 手工管理容易造成IP地址冲突
    3. 地址不能及时回收造成浪费

    2. 方案简介¶

    建立IP地址全生命周期管理，从地址资产规划拆分开始按业务、区域、类型等划分地址资产，构建地址资产规划使用流程管理，实现地址申请、地址工单上报、工单审批、资产自动化下发、使用、回收、分权分域管控自有网段，从而形成一体化的全流程管控能力。

    3. 核心优势¶

    1. 动静态IP统一管理
    2. 多维度智能分配
    3. 友好图形界面化

    4. IPAM介绍（监控易）¶

    4.1. 监控管理¶

    监控易IP地址管理可以展示IP地址与MAC地址的对应关系， 可以查看在用、未用的地址，也可以查看地址的分配、保留状况。

    4.2. 操作使用¶

    IP 地址可以手动添加、扫描或批量导入。

    4.3. 使用情况¶

    可以查看IP地址使用详情，IP 地址详情面板中列出所有可用IP的状态和类型，以及是否30天内活跃。

    4.4. 统计报告¶

    可以提供 IP 地址统计报告 包括规划网段、规划子网、可用IP、已用IP、未用IP、分配IP、管理IP、保留IP等。

    4.5. 子网使用率Top10¶

    子网使用率Top10：按照使用率从高到低进行排序。包括IP总数、已用IP数量、未用IP数量和IP使用率

    5. IP地址生命周期管理（）¶

    基于IP地址规划、分配、使用、备案、下线回收五个过程进行管理，形成完整的流程闭环。解决IP管理混乱，定位跟踪困难， 信息无法追溯等问题，确保IP地址最优化科学利用。

    在线IT资产统计分析：咨元IPAM可以自动发现网络设备及其端口和关系，并将这些信息同步给IT资产管理系统， 供IT资产管理部门及时掌握IT资产的在线信息； 同时IT资产管理系统中资产的编号、分类、位置、生产厂家、负责人、运维厂家等信息同步给IP地址管理系统， 便于网络运维部门通过IPAM系统一键调用查询。 采用咨元报表系统，可以对在线IT资产进行详细统计分析。

    1、地址规划： 在IPAM+地址管理平台中，实行分级分权管理。

    在总部级（总平台或总部管理员），进行全集团的IPv4/IPv6地址规划，总部单位的地址分配，IP地址管理系统维护和管理。在分支级（分平台或分支管理员），对本地的IPv4/IPv6进行详细的地址规划。

    最终创建详细的、规范的IPv6地址整体规划方案，实现对组织、业务、地域、网络等有效标识，便于实现统一的地址监控、溯源和管理。

    2、地址申请： 当需要新的业务上线，需由各分支业务部门或其他地址使用部门在系统中发起地址/地址段申请。先由各分支针对本地各业务进行不同地址段的规划，再由各分支IP地址管理人员进行审批，系统将预分配或手动分配具体的IPv4/IPv6地址。

    3、地址分配： 系统根据申请信息自动预分配或人工进行IPv4/IPv6地址的分配，分配后，将申请分配信息形成IP地址的备案信息，记录到系统中，为全集团IP地址统计分析、地址信息报备、IPv4/IPv6上报提供数据支持。

    4、地址使用： 地址上线使用过程中，平台将对地址使用情况进行跟踪监测。不管是办公、无线类的终端动态地址，还是网络、平台、专网类静态地址，平台将通过地址自动发现功能，采集网络中已使用的地址情况。

    使用的IP数据获取以后，将与规划数据进行审计，对异常IP产生告警，通过审计分析，逐步进行调整，提高网络质量，保障数据准确性。

    5、地址回收： 针对老旧业务下线，或者业务变更，需要进行回收/变更申请，将原有的地址/地址段进行回收，回收后，保障下次循环使用，及时变更IP分配数据，IP回收后，之前的分配地址数据进入系统历史记录，保障后期可以对地址进行跟踪回溯，同时提高IP使用效率，减少资源浪费，形成IP地址管理的流程闭环。

    最终形成从地址规划、申请、分配、追踪、回收的IP地址全生命周期的管理过程。

    6. IPAM（ZCDNS）¶

    信息化技术高速发展，IP资产管理手段亟待升级

    难溯源 : 万物互联时代，海量终端接入让IP管理、溯源困难重重。

    威胁多 : 陈旧、低效的 IP 管理方式造成私接路由泛滥，IP 冲突不断。

    效率低 : 大量企业仍旧随意通过交换机、自建Linux或Windows进行动态地址分配，出现问题只能人工查找。

    6.1. 八大功能¶

    1. 网络空间规划

      • 网络地址空间规划可视化
      • 同时支持多种IP地址类型可灵活定义使用
      • 支持以语义模板方式按位宽规划网络，大大降低IPv6网络管理难度
      • 支持规划数据导入导出，方便二次维护

    2. 智能DHCP服务

      • 支持动态静态IP地址分配、绑定式地址分配
      • 支持首次动态分配后续固定分配，大幅降低固定IP管理成本
      • 通过HA+failover技术确保服务稳定运行
      • 同时支持ZWP、MACOption、指纹等多种形式的对地址进行准入授权管理

    3. 地址动态感知

      • 支持终端IP地址资产分类管理
      • 支持多维度资产分析，建立清晰资产画像

    4. 终端资产管理

      • 支持IP地址周期扫描，主动进行IP地址发现
      • 支持多种扫描协议如ICMP、ARP、NetBios、Nmap

    5. 网络设备联动

      • 支持与外部网络设备联动并进行周期性扫描，获取IP地址等信息
      • 支持联动交换机进行终端IP/MAC/接口绑定，对风险终端及时阻断

    6. 地址历史全周期回溯

      • 状态变化
      • 信息变更
      • 告警事件地址回收
      • 自定义任意历史时间
      • 支持CSV格式导出

    7. 三方系统集成

      • AD域控
      • CMDB系统
      • 准入认证系统

    8. 多维数据报表

      • 支持网络使用率、地址池使用率、报文类型统计、LPS统计等数据报表
      • 基于特定业务需要，支持各产品各元素组合的定制化报表
      • 支持周期推送和CSV格式导出

    6.2. 三大优势¶

    1. 可视化管理体验 管理效率提升10倍以上
    2. 流程化排障溯源 排障时间缩短10倍以上
    3. IPv6顶层设计指导 管理难度降低10倍以上

    6.3. 四大价值¶

    1. 排障可溯 通过IP地址回放引擎实现IP溯源一站式服务，故障判定三方鉴责、溯源到人，显著提升用户处理故障效率。

    2. 资产可管 基于精准的资产指纹识别能力和配套的探针引擎掌握资产全貌，及时发现异常资产和潜在威胁。

    3. 网络可视 通过结构化的网络地址规划，智能收集全网IP地址关联信息，形成动态统一的网络台账，消除传统管理方式误区，提高管理效率。

    4. 风险可控 ”一键绑定””一键阻断”，通过与网络设备的深度联动，大幅降低网内潜在风险的暴露，提升快速响应和处置的能力。
