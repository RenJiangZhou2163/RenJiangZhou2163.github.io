    内网直播方案

    搭建公司内部直播平台可以通过使用开源软件实现，这样不仅可以节省成本，还可以根据公司的具体需求进行定制。以下是一个基于开源软件搭建公司内部直播平台的步骤：

    1. 选择合适的开源软件¶

    选择合适的开源直播软件是第一步。以下是一些常用的开源直播解决方案：

    • OBS Studio: 一个免费的开源视频录制和直播软件。
    • Nginx + RTMP Module: Nginx 是一个高性能的 HTTP 和反向代理服务器，RTMP 模块可以用于实时流媒体传输。
    • Red5: 一个开源的媒体服务器，支持流媒体、聊天、共享白板等功能。
    • Jitsi: 一个开源的视频会议解决方案，也支持直播功能。

    2. 搭建基础环境¶

    需要准备一台服务器，可以使用云服务器或公司内部的物理服务器。

    安装 Nginx 和 RTMP 模块¶

    # 安装依赖
sudo apt-get update
sudo apt-get install build-essential libpcre3 libpcre3-dev libssl-dev

# 下载 Nginx 和 RTMP 模块源码
wget http://nginx.org/download/nginx-1.20.1.tar.gz
wget https://github.com/arut/nginx-rtmp-module/archive/master.zip

# 解压源码包
tar -zxvf nginx-1.20.1.tar.gz
unzip master.zip

# 编译和安装 Nginx 与 RTMP 模块
cd nginx-1.20.1
./configure --with-http_ssl_module --add-module=../nginx-rtmp-module-master
make
sudo make install

# 启动 Nginx
sudo /usr/local/nginx/sbin/nginx

    3. 配置 Nginx 的 RTMP 模块¶

    编辑 nginx.conf 文件：

    sudo vim /usr/local/nginx/conf/nginx.conf

    添加以下配置：

    rtmp {
    server {
        listen 1935;
        chunk_size 4096;

        application live {
            live on;
            record off;

            # 推流地址 (推送到这个 URL)
            push rtmp://localhost/live;
        }
    }
}

http {
    server {
        listen 8080;

        location / {
            root html;
            index index.html index.htm;
        }

        location /hls {
            types {
                application/vnd.apple.mpegurl m3u8;
                video/mp2t ts;
            }
            alias /tmp/hls;
            expires -1;
        }

        location /stat {
            rtmp_stat all;
            rtmp_stat_stylesheet stat.xsl;
        }

        location /stat.xsl {
            root html;
        }
    }
}

    4. 安装和配置 OBS Studio¶

    OBS Studio 可以在各大操作系统上安装（Windows, MacOS, Linux），安装后进行如下配置：

    1. 打开 OBS Studio，点击“设置”。
    2. 在“流”选项卡中，选择“自定义流媒体服务”。
    3. 输入 RTMP URL，例如：rtmp://<your_server_ip>/live。
    4. 设置好其他参数后，开始推流。

    5. 播放直播内容¶

    可以使用 VLC 或者网页播放器播放直播内容。例如，可以在 VLC 中打开网络串流：rtmp://<your_server_ip>/live。

    也可以在网页中嵌入播放器，例如 Video.js：

    <!DOCTYPE html>
<html>
<head>
    <link href="https://vjs.zencdn.net/7.11.4/video-js.css" rel="stylesheet" />
</head>
<body>
    <video id="my_video_1" class="video-js vjs-default-skin" controls preload="auto" width="640" height="264"
           data-setup='{"fluid": true}'>
        <source src="http://<your_server_ip>:8080/hls/stream.m3u8" type="application/x-mpegURL">
    </video>

    <script src="https://vjs.zencdn.net/7.11.4/video.js"></script>
</body>
</html>

    总结¶

    通过上述步骤，你可以基于 Nginx 和 OBS Studio 搭建一个简单而高效的公司内部直播平台。当然，根据具体需求，还可以进一步优化和扩展功能，如增加用户认证、聊天互动等功能。
